How to use
1. Pick a date
Use the date picker or quick Mon–Thu buttons. Template auto-loads the default schedule.
2. Build routes
Add properties from the dropdown or type custom names. Reorder with ↑↓. Remove with ✕.
3. Save & dispatch
Save per crew or click "Save All". Routes push live to crew phones instantly via Firestore.
